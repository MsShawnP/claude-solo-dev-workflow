# claude-solo-dev-workflow

A workflow for solo developers using Claude Code and Claude chat together. Templates, slash commands, and reference materials for running portfolio projects with structured state management, journaling, and decision tracking.

## What's here

- `workflow-package/` — the actual templates and reference files
- `workflow-package/README.md` — setup checklist for using this on a new project

## Origin

Built by Shawn Phillips for solo portfolio work — data analysis, reporting, visualization, and adjacent projects. Distilled from working in Claude Code across multiple projects. The workflow emphasizes:

- State in files, not in sessions
- Structured journaling at session end
- Explicit failure capture (not just success)
- Vertical slices over horizontal phases
- Measurable rules over vague guidance
